# Bubble Game

A fun and interactive Bubble Game built with HTML, CSS, and JavaScript. The game involves popping bubbles by clicking on them within a limited time, making it a challenging and enjoyable experience for players of all ages.

## Features

- **Bubble Generation**: Bubbles randomly appear on the screen with varying sizes and speeds.
- **Click to Pop**: Players click on the bubbles to pop them and score points.
- **Timer**: A countdown timer keeps track of the time remaining to pop as many bubbles as possible.
- **Score**: The score increases as bubbles are popped, and the game ends when the timer runs out.
